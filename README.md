# caixabank-prueba-tecnica-backend

Proyecto base Spring Boot (Java 17) para prueba técnica.

## Ejecutar
```bash
mvn spring-boot:run
```

## H2 Console
- URL: http://localhost:8080/h2-console
- JDBC URL: jdbc:h2:file:./caixabank_db
- User: sa
- Password: (vacío)

La base de datos se persiste en `./caixabank_db.mv.db` (en la raíz del proyecto).
